# wood_samples
This data set consists of 108 samples of timber,all from eighteen types of timber, six samples per type:
- Type 0 - Muiracatiara
- Type 1 - Jarana
- Type 2 - Macacauba
- Type 3 - Purpleheart
- Type 4 - Angelim rajada
- Type 5 - Wacapou
- Type 6 - Angelim pedra
- Type 7 - Manupatinga
- Type 8 - Cumaru
- Type 9 - Timborana
- Type 10 - Yellowheart
- Type 11 - Brazilwood
- Type 12 - Rim paca
- Type 13 - Marupa
- Type 14 - Pataua
- Type 15 - Araracanga
- Type 16 - Ceder
- Type 17 - Mahogany

The samples were digitalized with a common scanner (CanonG300,  300  dpi), and have the same size (391  x  165  pixels)
